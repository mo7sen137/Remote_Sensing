## 🔧 Customization Guide

### To Change Colors
Edit `/app/main.py` in the CSS section (lines 1-350):
```python
# Search for color hex codes and replace:
#00ff88 → Your accent color
#00ffff → Your header color
#ff00ff → Your secondary color
#0a0e27 → Your background color
```

### To Add Animations
Extend the keyframes section:
```python
@keyframes custom-name {
    0% { /* start */ }
    100% { /* end */ }
}
```

### To Modify Typography
Update the Google Fonts import and font sizes in CSS.
