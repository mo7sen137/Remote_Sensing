## 📍 المحاضرة الثانية: المنصات والأقمار والدقة

```
┌─────────────────────────────────────────────────────────────────┐
│         منصات الاستشعار الأساسية (Primary Platforms)          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌──────────────────────────────────────────────────────────┐  │
│  │ 1️⃣  المنصات الأرضية (Ground-Based) 🔭                 │  │
│  │    • موقع: الأرض                                         │  │
│  │    • التغطية: محدودة جداً                               │  │
│  │    • الدقة: عالية جداً                                   │  │
│  │    • التكلفة: رخيصة                                      │  │
│  │    • مثال: محطات الطقس                                  │  │
│  └──────────────────────────────────────────────────────────┘  │
│                          ▲                                       │
│                          │                                       │
│  ┌──────────────────────────────────────────────────────────┐  │
│  │ 2️⃣  المنصات الجوية (Aerial) ✈️                         │  │
│  │    • موقع: الطائرات                                     │  │
│  │    • التغطية: متوسطة                                    │  │
│  │    • الدقة: متوسطة لعالية                               │  │
│  │    • التكلفة: متوسطة                                    │  │
│  │    • مثال: الصور الجوية (Aerial Photography)            │  │
│  └──────────────────────────────────────────────────────────┘  │
│                          ▲                                       │
│                          │                                       │
│  ┌──────────────────────────────────────────────────────────┐  │
│  │ 3️⃣  المنصات الفضائية (Spaceborne) 🛰️                 │  │
│  │    • موقع: الأقمار الصناعية                             │  │
│  │    • التغطية: عالمية ضخمة                              │  │
│  │    • الدقة: منوعة (من 30m لـ 0.5m)                     │  │
│  │    • التكلفة: غالية جداً                                │  │
│  │    • مثال: Landsat, Sentinel, Spot                      │  │
│  └──────────────────────────────────────────────────────────┘  │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│              المدارات الرئيسية (Main Orbits) 🪐                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  🔴 مدار ثابت بالنسبة للأرض (GEO)                             │
│     └─ الارتفاع: 36,000 كم | الدقة: 1 كم | مثال: الأرصاد     │
│                                                                 │
│  🟢 مدار شبه قطبي (Near-Polar)                                 │
│     └─ يمر قطرياً | الدقة: عالية | مثال: Landsat 8           │
│                                                                 │
│  🔵 مدار متزامن مع الشمس (SSO)                                │
│     └─ يحافظ على زاوية إضاءة ثابتة | الدقة: عالية           │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│         أنواع الدقة (Types of Resolution)                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  📏 الدقة المكانية (Spatial Resolution)                        │
│     • حجم البكسل الواحد (الـ pixel size)                       │
│     • Landsat 8: 30 متر                                        │
│     • WorldView: 0.3 متر (عالية جداً)                         │
│                                                                 │
│  🌈 الدقة الطيفية (Spectral Resolution)                        │
│     ├─ بانكروماتي (Panchromatic): حزمة واحدة (رمادي)        │
│     ├─ متعدد الأطياف (Multispectral): 3-10 حزم 🎨            │
│     └─ فائق الأطياف (Hyperspectral): 100+ حزمة 🔬            │
│                                                                 │
│  💫 الدقة الإشعاعية (Radiometric Resolution)                   │
│     • عمق الألوان (bit depth)                                  │
│     • 8-bit: 256 مستوى (عادي)                                 │
│     • 12-bit أو أعلى (دقيق)                                  │
│                                                                 │
│  ⏰ الدقة الزمنية (Temporal Resolution)                        │
│     • كم مرة يمر القمر على نفس المكان                         │
│     • Landsat 8: كل 16 يوم                                    │
│     • Sentinel 2: كل 5 أيام                                   │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│            أنواع المستشعرات (Types of Sensors)                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  📷 الكاميرات والتصوير الجوي (Cameras)                         │
│  📊 المسح متعدد الأطياف (MSS - Multispectral Scanning)       │
│  🌡️  التصوير الحراري (Thermal Imaging)                       │
│  📡 الرادار (Radar - في المحاضرة الثالثة)                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 📍 المحاضرة الثالثة: استشعار الميكروويف و الرادار

```
┌─────────────────────────────────────────────────────────────────┐
│        استشعار الميكروويف (Microwave Remote Sensing)           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 🔴 الاستشعار السلبي (Passive)                         │   │
│  │    • الجهاز: الراديومتر (Radiometer)                  │   │
│  │    • المبدأ: يستقبل الطاقة من الهدف                  │   │
│  │    • التطبيق:                                         │   │
│  │      ├─ الأرصاد الجوية ☁️                           │   │
│  │      ├─ الهيدرولوجيا 💧                             │   │
│  │      ├─ علم المحيطات 🌊                             │   │
│  │      └─ دراسات بيئية 🌍                             │   │
│  └─────────────────────────────────────────────────────────┘   │
│                          ▲                                       │
│                          │                                       │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 🟢 الاستشعار الإيجابي (Active)                         │   │
│  │    • الجهاز: الرادار (RADAR - Radio Detection)        │   │
│  │    • المبدأ: يرسل إشارة ويستقبلها                    │   │

│  │      └─ غير تصويري (Non-Imaging):                    │   │
│  │         ├─ مقاييس الارتفاع (Altimeters)              │   │

│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│          نطاقات تردد الرادار (Radar Frequency Bands)           │
├─────────────────────────────────────────────────────────────────┤
│                 (من الأطول موجة للأقصر)                        │
│                                                                 │
│  🔴 P-band  0.3−0.4 GHz    (أطول موجة)                        │
│  🟠 L-band  1−2 GHz                                             │
│  🟡 S-band  2−4 GHz                                             │
│  🟢 C-band  4−8 GHz         (الأكثر استخداماً)               │
│  🔵 X-band  8−12 GHz                                            │
│  🟣 Ku-band 12−18 GHz                                           │
│  ⚪ K-band  18−27 GHz         (قصيرة جداً)                     │
│  ⬛ Ka-band 27−40 GHz                                           │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│           هندسة التصوير الراداري (Radar Geometry)             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│     ╔════════════════════════════════════════════════╗         │
│     ║  هوائي الرادار                                 ║         │
│     ║        │                                         ║         │
│     ║        │ حزمة الموجات                           ║         │
│     ║        ▼                                         ║         │
│     ║    ╱╲╱╲╱╲╱╲╱╲                                   ║         │
│     ║   ╱          ╲                                   ║         │
│     ║  ╱    السمت  ╲  (Azimuth - along-track)        ║         │
│     ║ ╱    (X-axis) ╲                                 ║         │
│     ║╱────────────────╲────────────────────────────  ║         │
│     ║        المدى (Range - across-track)            ║         │
│     ║        (Y-axis)                                 ║         │
│     ║                                                  ║         │
│     ║  المدى القريب       المدى البعيد               ║         │
│     ║  (Near Range)    (Far Range)                   ║         │
│     ║   ◄────────────────────────►                   ║         │
│     ╚════════════════════════════════════════════════╝         │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│       الاستقطاب (Polarization) والتركيبات                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────────────────────────────┐                        │
│  │ متشابه (Like-Polarized)             │                        │
│  │ ─────────────────────  H → HH       │                        │
│  │ الإرسال والاستقبال   │              │                        │
│  │ بنفس الاتجاه          V → VV       │                        │
│  └─────────────────────────────────────┘                        │
│                       AND                                       │
│  ┌─────────────────────────────────────┐                        │
│  │ متقاطع (Cross-Polarized)            │                        │
│  │ ─────────────────────  H → HV       │                        │
│  │ الإرسال والاستقبال   │              │                        │
│  │ باتجاهات مختلفة       V → VH       │                        │
│  └─────────────────────────────────────┘                        │
│                                                                 │
│  التطبيق:                                                       │
│   • HH/VV: أفضل لرؤية الأرضية                                 │
│   • HV/VH: أفضل لرؤية الغطاء النباتي                          │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│          مشاكل وخصائص صور الرادار (Issues & Properties)       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ⚠️  ضوضاء البقع (Speckle Noise) 🎲                           │
│     السبب: تداخل الأمواج المنعكسة من نقاط متعددة            │
│     الحل:                                                       │
│     ├─ المعالجة متعددة الرؤى (Multi-look processing)         │
│     └─ التصفية المكانية (Spatial filtering)                   │
│                                                                 │
│  ⚠️  تشوه المدى المائل (Slant Range Distortion)              │
│     الصورة مشوهة بسبب الزاوية المائلة                        │
│                                                                 │
│  ⚠️  تأثير نمط الهوائي (Antenna Pattern Effect)              │
│     قوة الإشارة تختلف حسب زاوية الهوائي                      │
│                                                                 │
│  ✅ الدقة الإشعاعية والنطاق الديناميكي                        │
│     • 8-bit عادة في معظم الأقمار الرادارية                   │
│                                                                 │
│  ✅ معايرة الرادار (Radar Calibration)                        │
│     ├─ نسبية (Relative): بين صور مختلفة                     │
│     └─ مطلقة (Absolute): قيمة فيزيائية حقيقية             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 📊 جدول المقارنة الشامل

| المعيار | المحاضرة الأولى | المحاضرة الثانية | المحاضرة الثالثة |
|--------|-----------------|------------------|------------------|
| **المحور الرئيسي** | الأساسيات والفيزياء | المنصات والتجهيزات | تقنية الرادار |
| **المدى الموجي** | UV - Microwave | جميع الأنواع | Microwave فقط |
| **العمل ليلاً** | ❌ بعضها فقط | ⚠️ حراري فقط | ✅ نعم |
| **النفاذ عبر السحب** | ❌ لا تخترق | ❌ لا تخترق | ✅ نعم تخترق |
| **تطبيقات عملية** | الفهرسة النظرية | اختيار الأقمار | الكشف الهندسي |
| **المستشعرات** | عامة | Landsat, Sentinel | SAR, Radar |
