## نظام معالجة البيانات الجديد

### الملفات الرئيسية المضافة

#### 1️⃣ `src/model_trainer.py` (الملف الأساسي الجديد)
ملف شامل يحتوي على 4 فئات رئيسية:

**`FeatureExtractor`** - استخراج الخصائص الطيفية
```python
# معالجة الموجات الراديومترية
B_cal = FeatureExtractor.calibrate_bands(bands_data)

# حساب المؤشرات الطيفية (NDVI, NDWI, NDBI)
NDVI, NDWI, NDBI = FeatureExtractor.calculate_indices(B_cal)

# تجميع الخصائص (7 bands + 3 indices = 10 features)
features = FeatureExtractor.create_feature_stack(B_cal, NDVI, NDWI, NDBI)
```

**`ModelTrainer`** - تدريب النماذج
```python
trainer = ModelTrainer()

# تحضير بيانات التدريب من ROI
X_train, X_test, Y_train, Y_test = trainer.prepare_training_data(
    roi_df, B_cal, NDVI, NDWI, NDBI
)

# تدريب جميع النماذج
results = trainer.train_all_models(X_train, X_test, Y_train, Y_test)

# الحصول على الإحصائيات
stats_df = trainer.get_statistics_dataframe()
```

**`ClassificationMapper`** - إنشاء الخرائط والإحصائيات
```python
# إنشاء خريطة التصنيف الملونة
color_map = ClassificationMapper.create_colored_map(class_map)

# حساب إحصائيات المساحة
area_stats = ClassificationMapper.calculate_area_statistics(class_map)
```

### تعديلات `app/main.py`

#### صفحة الرفع الجديدة (`page_upload`)
**تم التغيير من:**
- 7 ملفات منفصلة (GeoTIFF)
- 1 ملف MTL

**إلى:**
- 1 ملف CSV (يحتوي على جميع الباندات)
- 1 ملف BIN (البيانات الوصفية)

#### صفحة التصنيف المحسّنة (`page_classification`)
```python
# المراحل الجديدة:
1. تحميل البيانات من CSV
2. معايرة راديومترية
3. حساب المؤشرات الطيفية
4. تحضير بيانات التدريب من ROI
5. تدريب النموذج المختار
6. توليد خريطة التصنيف
7. حساب الإحصائيات
```

#### صفحة النتائج المحديثة (`page_results`)
- عرض دقة النموذج (Training & Testing)
- خريطة التصنيف الملونة
- إحصائيات المساحة والنسب المئوية
- خيارات التحميل (PNG, CSV, TXT)

### متطلبات الملفات

#### 1. ملف البيانات (CSV)
```
B1,B2,B3,B4,B5,B6,B7,...
1000,1200,1100,1050,2000,1500,800,...
1050,1250,1150,1100,2050,1550,850,...
```

#### 2. ملف البيانات الوصفية (BIN)
- ملف ثنائي يحتوي على معلومات الصورة
- الحد الأدنى: معرّف الصورة أو معلومات التاريخ

#### 3. ملف التدريب (ROI CSV)
```
B1,B2,B3,B4,B5,B6,B7,Class_Label
1000,1200,1100,1050,2000,1500,800,1
1050,1250,1150,1100,2050,1550,850,1
...
1800,1900,1950,1850,3000,2500,1200,2
```

**Class_Label قيم:**
- 1 = Water (ماء)
- 2 = Vegetation (نبات)
- 3 = Urban (حضر)
- 4 = Desert (صحراء)

### ملاحظات مهمة

1. **معايرة البيانات** - يتم تطبيقها تلقائياً باستخدام:
   - المعامل M = 2e-5
   - المعامل A = -0.1
   - زاوية الشمس الافتراضية = 39.31°

3. **تطبيع الخصائص** - يتم التطبيع للنماذج التي تتطلبها

4. **التصنيفات** - 4 فئات أرضية ثابتة

5. **حجم البيكسل** - 30 متر (معيار Landsat 8)
