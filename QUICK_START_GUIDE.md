## دليل الاستخدام السريع للنظام الجديد

### 🚀 البدء السريع

#### المتطلبات:
✓ ملف CSV للبيانات الطيفية (الباندات)
✓ ملف BIN للبيانات الوصفية (يمكن أن يكون ملف نصي فارغ للاختبار)
✓ ملف ROI CSV لبيانات التدريب

---

### 📋 خطوات الاستخدام

#### 1️⃣ تشغيل التطبيق
```bash
streamlit run app/main.py
```

#### 2️⃣ الذهاب إلى صفحة "Upload"
- **رفع البيانات الطيفية:**
  - انقر على `Upload Bands CSV`
  - اختر ملف CSV يحتوي على 7 أعمدة (B1-B7)

- **رفع البيانات الوصفية:**
  - انقر على `Upload Metadata File`
  - اختر ملف BIN (أو حتى ملف نصي)

#### 3️⃣ الذهاب إلى صفحة "Classification"
- اختر النموذج المطلوب من القائمة المنسدلة
- رفع ملف ROI CSV (بيانات التدريب)
  - يجب أن يحتوي على 8 أعمدة: B1-B7 + Class_Label
  - Class_Label يجب أن يكون: 1 (ماء)، 2 (نبات)، 3 (حضر)، 4 (صحراء)

#### 4️⃣ تنفيذ التصنيف
- اضغط على **"Execute Classification"**
- انتظر انتهاء العملية
- ستظهر رسالة نجاح

#### 5️⃣ عرض النتائج
- اذهب إلى صفحة **"Results"**
- ستجد:
  - دقة النموذج (التدريب والاختبار)
  - خريطة التصنيف الملونة
  - إحصائيات المساحة
  - خيارات التحميل

### 🔍 فهم البيانات

#### ملف CSV البيانات الطيفية (Bands)
```
الصف الأول: رؤوس الأعمدة (B1, B2, ..., B7)
الصفوف التالية: قيم الموجات لكل بيكسل

مثال:
B1,B2,B3,B4,B5,B6,B7
5000,4800,4600,4400,6500,4000,3800
5100,4900,4700,4500,6600,4100,3900
...
```

**نطاق القيم المتوقعة:**
- عادةً بين 0-10000 (لصور Landsat 8)
- النظام يعاير هذه القيم تلقائياً

#### ملف BIN البيانات الوصفية
- يمكن أن يكون أي ملف ثنائي
- للاختبار: يمكن استخدام ملف نصي أو حتى ملف فارغ
- في الإنتاج: سيحتوي على تاريخ الاستحواذ، موقع الاستقراء، إلخ

#### ملف CSV التدريب (ROI)
```
الأعمدة 1-7: نفس ترتيب الباندات (B1-B7)
العمود 8: التصنيف (Class_Label)

Class_Label:
- 1 = Water (ماء)
- 2 = Vegetation (نبات)
- 3 = Urban (حضر)
- 4 = Desert (صحراء)

مثال:
B1,B2,B3,B4,B5,B6,B7,Class_Label
3000,2800,2600,2400,5000,2500,1800,1
7000,6500,4500,2000,8000,6000,5000,2
8500,7000,5000,3000,3000,8000,7500,3
2500,2300,2100,1900,2000,3000,4000,4
```

### 🎯 نصائح لأفضل النتائج

1. **حجم عينات التدريب:**
   - الحد الأدنى: 10 عينات لكل فئة
   - الأمثل: 30-50 عينة لكل فئة
   - في sample_roi.csv: 9 عينات لكل فئة (للاختبار فقط)

2. **جودة البيانات:**
   - اتأكد من عدم وجود قيم ناقصة (NaN)
   - تجنب القيم الشاذة

3. **اختيار النموذج:**
   - **Random Forest:** الأفضل للبيانات الصغيرة والمتوسطة ⭐
   - **SVM:** جيد عندما تكون البيانات محدودة
   - **MLP:** الأفضل مع كميات بيانات كبيرة

4. **معايرة البيانات:**
   - تتم تلقائياً باستخدام معاملات Landsat 8
   - إذا كانت بيانات أخرى، قد تحتاج لتعديل المعاملات

### 🔧 استكشاف الأخطاء

#### مشكلة: "Error loading CSV"
**الحل:** تأكد أن:
- ✓ الملف بصيغة CSV
- ✓ يحتوي على 7 أعمدة على الأقل
- ✓ لا توجد أسطر فارغة

#### مشكلة: "Invalid ROI data"
**الحل:** تأكد من:
- ✓ العمود الأخير اسمه "Class_Label"
- ✓ القيم 1، 2، 3، 4 فقط
- ✓ عدد الأعمدة = 8

#### مشكلة: "No data available"
**الحل:**
- تأكد من رفع البيانات قبل الضغط على Execute
- افحص رسائل الخطأ في الكونسول
